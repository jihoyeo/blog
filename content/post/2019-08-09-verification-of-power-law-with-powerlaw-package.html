1. Discrete data: The Moby Dick data set

The Moby Dick data set contains the frequency of unique words in the novel Moby Dick by Herman Melville. This data set can be downloaded from http://tuvalu.santafe.edu/вҲјaaronc/powerlaws/data.htm or loaded directly

library("poweRlaw")
## Warning: нҢЁнӮӨм§Җ 'poweRlaw'лҠ” R лІ„м „ 3.5.2м—җм„ң мһ‘м„ұлҗҳм—ҲмҠөлӢҲлӢӨ
data("moby", package="poweRlaw")

To fit a discrete power law to this data, we use the displ constructor

m_pl <- displ$new(moby)

We estimate the lower threshold via

est <- estimate_xmin(m_pl)

and update the power law object

m_pl$setXmin(est)

For a given value xmin, the scaling parameter is estimated by numerically optimising the loglikelihood. The optimiser is initialised using the analytical MLE

\(\hat{\alpha} = 1 + n[\sum_{i=1}^n log({x_i\over x_{min}-0.5})]^{-1}\)

Parameter uncertainty

To get a handle on the uncertainty in the parameter estimates, we use a bootstrapping procedure, via the bootstrap function.

## 5000 bootstraps using two cores
bs = bootstrap(m_pl, no_of_sims=5000, threads=2)
